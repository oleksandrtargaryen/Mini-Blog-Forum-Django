# Mini-Blog-Forum-Django
Mini Blog – простий блог на Django. Є логін/логаут через Django auth, CRUD постів, коментарі та адмінка. Використання session-based auth з CSRF захистом. Щоб запустити: клонувати репозиторій, перейти в папку, створити віртуальне середовище, активувати його, встановити Django та psycopg2-binary, виконати міграції, створити суперюзера і запустити сервер. URL: / – список постів, /create/ – створення поста, /<post_id>/ – перегляд поста і коментарі, /accounts/login/ – логін, /accounts/logout/ – логаут, /admin/ – адмінка. Моделі: Post (title, body, author, created_at), Comment (post, body, author, created_at).
